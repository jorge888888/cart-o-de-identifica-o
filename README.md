# Cartão de Identificação Mobile

Aplicativo mobile desenvolvido com Ionic Framework + Angular durante 
o curso de Análise e Desenvolvimento de Sistemas.

## Tecnologias
- Ionic Framework
- Angular
- TypeScript
- SCSS
- Capacitor (Android/iOS)

## Como executar
# Instalar dependências
npm install

# Rodar no navegador
ionic serve

# Gerar build para Android
ionic capacitor build android
